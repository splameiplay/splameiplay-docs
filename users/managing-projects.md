# Managing projects

## Installing a project (native)

To install a project, first open the project page to the screen below

<figure><img src="../.gitbook/assets/SplameiPlay by Splamei - 11.8.3 - 18.1.2025.png" alt=""><figcaption><p>The Bitorpito Deluxe Project Page</p></figcaption></figure>

Press download and SplameiPlay will download and install the project for you. SplameiPlay may freeze while installing if you're on an older version of the app.

## Installing a project (custom)

To learn how to install a custom projects, see the [custom projects](custom-projects.md) page.

## Opening a project

When on the project page, all you have to do is press 'Play'. SplameiPlay will automatically check for updates, update the project if there is any, otherwise it will open the project for you.

## Uninstalling a project

To uninstall a custom project, scroll down to the end of the page. Press 'Uninstall' and confirm you want to uninstall the project.

If the project was a custom file or native, the project will be completed deleted and all SplameiPlay data about it deleted except for data used by recap. If the project was added through settings, only SplameiPlay's data will be deleted.
