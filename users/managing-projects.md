# Managing projects

## Installing a project (native)

To install a project, first open the project page to the screen below

<figure><img src="../.gitbook/assets/SplameiPlay by Splamei - 11.8.3 - 18.1.2025.png" alt=""><figcaption><p>The Bitorpito Deluxe Project Page</p></figcaption></figure>

Press download and SplameiPlay will download and install the project for you. SplameiPlay will freeze but this won't be for long.

## Installing a project (custom)

To know how to install a custom projects, see the [custom projects](custom-projects.md) page.

## Opening a project

When on the project page, all you have to do is press 'Play'. SplameiPlay will then check for updates automatically, update if there are any, then open the project so that your ready to play

## Uninstalling a project

To uninstall a custom project, scroll down until you find a screen similar to the one below

<figure><img src="../.gitbook/assets/Project Page 2.png" alt=""><figcaption><p>The Bitorpito Deluxe Project Page but scrolled down</p></figcaption></figure>

Press 'Uninstall' and confirm you want to uninstall the project. If the project was a custom file or native, the project will be completed deleted and all SplameiPlay data about it deleted except for data used by recap. If the project was added through settings, only SplameiPlay's data will be deleted
